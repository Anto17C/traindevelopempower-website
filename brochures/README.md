# TDE Course Brochure Files

This folder contains the production PDF and editable HTML/CSS source for Course 1.

## Live PDF used by the website

- `scl-resilience-brochure.pdf`

The existing course page download button should point to:

```text
/brochures/scl-resilience-brochure.pdf
```

## Editable source

- `scl-resilience-brochure.html`
- `assets/brochure.css`

The source uses existing website images already present in `/images/`, including:

- `/images/logo.svg`
- `/images/logo-primary.svg`
- `/images/scl-resilience-hero.webp`
- `/images/scl-resilience-faq.webp`

The PDF itself is standalone for website visitors. It does not need CSS or image files to display.
The HTML/CSS files are included so the brochure can be edited or regenerated later.
